# winapps-utils

Desktop utilities for managing a [WinApps](https://github.com/winapps-org/winapps) Windows-in-Docker
setup, without typing commands.

Each utility lives in its own folder with a `README.md` and an `install.sh`. Clone the repo, then
install whichever utils you want:

```bash
git clone https://github.com/LavakumarVeludandi/winapps-utils.git
cd winapps-utils/<util-name>
./install.sh
```

## Utils

| Util | Description |
| --- | --- |
| [vm-controller](vm-controller/) | GUI menu (Start/Pause/Unpause/Restart/Stop/Kill) for the WinApps VM, with desktop notifications. |
